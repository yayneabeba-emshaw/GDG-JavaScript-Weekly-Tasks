# GDG JavaScript Weekly Tasks

This repository contains my weekly practice tasks for learning **HTML, CSS, and JavaScript**.
Each week focuses on different concepts to improve programming and problem-solving skills.

# Week 1 – JavaScript Basics

### Task  – Online Shopping Calculation

A program that calculates the final payment amount for a customer.

**Features**

* Checks if the customer is eligible for a regional discount.
* Applies a fixed discount for the **East region**.
* Calculates **8% sales tax**.
* Displays:

  * Tax amount
  * Discount
  * Final payment amount.

**Concepts Used**

* Variables
* Conditional statements (`if`)
* Basic arithmetic operations
* Console output

---

### Task 1– Grade Calculator

A program that takes marks from the user and calculates the grade.

**Grade Rules**

* 90–100 → A+
* 85–89 → A
* 80–84 → A-
* 75–79 → B+
* 70–74 → B
* 65–69 → B-
* 60–64 → C+
* Below 60 → F

**Concepts Used**

* `prompt()` for user input
* `if / else if` conditions
* Input validation
* Console output

---

### Task 2 – Simple Login System

A basic login system that verifies user credentials.

**Rules**

* Username must be **admin**
* Password must be **1234**

**Output**

* If correct → *Login Successful*
* Otherwise → *Invalid Credentials*

**Concepts Used**

* Conditional logic
* Logical operators (`&&`)
* Input validation

# Week 2 – Functions and Loops

### Number Processing Program

The program loops through numbers **5 to 15** and performs several checks.

**Program Tasks**

* Identify whether a number is **odd or even**
* Check if it is **divisible by 3, 5, or both**
* Calculate the **total sum of numbers**
* Count the **odd numbers**
* Count the **even numbers**

**Concepts Used**

* `for` loop
* Functions
* Modulus operator `%`
* Counters and accumulators

Functions created:

* `checkOddEven()`
* `checkDivisibility()`
* `addToSum()`
* `processNumbers()`
